# heatmap
heatmap.js is a plugin for canvas chart 一个基于canvas的绘制热图插件

### 概述
工作中我们使用[heatmap.js](https://www.patrick-wied.at/static/heatmapjs/)这个热图绘制工具,然后业务的风向变了, 
就阅读源码学习, 重新一套代码适应需求变化, 除了绘制原理是抄它的之外, 其他都是自己写的.
